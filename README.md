# NGR Fork of Gorilla WebSocket

[Original pkg repository](https://github.com/gorilla/websocket)

Gorilla WebSocket is a [Go](http://golang.org/) implementation of the
[WebSocket](http://www.rfc-editor.org/rfc/rfc6455.txt) protocol.


### Documentation

* [Original pkg API Reference](https://pkg.go.dev/github.com/gorilla/websocket?tab=doc)
* [Chat example](https://github.com/NGRsoftlab/ngr-websocket/tree/main/examples/chat)
* [Command example](https://github.com/NGRsoftlab/ngr-websocket/tree/main/examples/command)
* [Client and server example](https://github.com/NGRsoftlab/ngr-websocket/tree/main/examples/echo)
* [File watch example](https://github.com/NGRsoftlab/ngr-websocket/tree/main/examples/filewatch)

### Status

The Gorilla WebSocket package provides a complete and tested implementation of
the [WebSocket](http://www.rfc-editor.org/rfc/rfc6455.txt) protocol. The
package API is stable.

### Installation

    go get github.com/NGRsoftlab/ngr-websocket

### Protocol Compliance

The Gorilla WebSocket package passes the server tests in the [Autobahn Test
Suite](https://github.com/crossbario/autobahn-testsuite) using the application in the [examples/autobahn
subdirectory](https://github.com/NGRsoftlab/ngr-websocket/tree/main/examples/autobahn).
